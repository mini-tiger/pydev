

# 示例Markdown文档

## 第一章：引言

Markdown是一种轻量级的标记语言，可以转换成HTML等格式。它由John Gruber创建，并且深受程序员和作家们的欢迎。

### 什么是Markdown？

Markdown是一种纯文本格式的标记语言，通过简单的标记符号，用户可以生成丰富的文本效果，比如标题、列表、引用等。

## 第二章：Markdown语法

Markdown语法非常简单，以下是一些常用的Markdown语法示例。

### 标题

Markdown支持多级标题，使用井号（`#`）符号表示。

```markdown
# 一级标题
## 二级标题
### 三级标题
```
